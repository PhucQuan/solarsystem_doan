# Planet Textures

## Hướng dẫn tải textures cho hành tinh

### Nguồn 1: Solar System Scope (Khuyên dùng - Free & Chất lượng cao)
Link: https://www.solarsystemscope.com/textures/

Tải các file sau (chọn độ phân giải 2K):
- `2k_sun.jpg` → đổi tên thành `sun.jpg`
- `2k_mercury.jpg` → đổi tên thành `mercury.jpg`
- `2k_venus_surface.jpg` → đổi tên thành `venus.jpg`
- `2k_earth_daymap.jpg` → đổi tên thành `earth.jpg`
- `2k_earth_clouds.jpg` → đổi tên thành `earth_clouds.jpg` (optional)
- `2k_mars.jpg` → đổi tên thành `mars.jpg`
- `2k_jupiter.jpg` → đổi tên thành `jupiter.jpg`
- `2k_saturn.jpg` → đổi tên thành `saturn.jpg`
- `2k_uranus.jpg` → đổi tên thành `uranus.jpg`
- `2k_neptune.jpg` → đổi tên thành `neptune.jpg`

### Nguồn 2: NASA (Textures chính thức)
Link: https://science.nasa.gov/resource/solar-system-and-beyond-poster-set/

### Nguồn 3: Planet Pixel Emporium
Link: http://planetpixelemporium.com/planets.html

## Sau khi tải xong
Đặt tất cả file .jpg vào thư mục này (`public/textures/`)

Cấu trúc thư mục sẽ như sau:
```
public/
  textures/
    sun.jpg
    mercury.jpg
    venus.jpg
    earth.jpg
    earth_clouds.jpg
    mars.jpg
    jupiter.jpg
    saturn.jpg
    uranus.jpg
    neptune.jpg
```

## Lưu ý
- Nếu không có texture, hệ thống sẽ tự động dùng màu placeholder
- Độ phân giải 2K (2048x1024) là đủ cho web
- File size mỗi ảnh khoảng 500KB - 2MB
